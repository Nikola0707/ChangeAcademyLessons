HTML Colors

Understanding Color in HTML

HTML allows you to specify colors using various methods:

  • Color Names: HTML supports a set of predifined color names such as red, blue and green
  • Hexadecimal Notation: Colors can also be specified using hecadecimal notation, such as #FF0000 for red and #008000 for green.
  • RGB Values: RBA (Red, Green, Blue) calues can be used to define colors, such as rgb(255, 0, 0) for red and rgb(0, 128, 0) for green.
  • HSL Values: HSL (Hue, saturation, Lightness) values provide another way to specify colors such a hsl(240, 100%, 50%) for blue and hsl(120, 100%, 25%) for green.
